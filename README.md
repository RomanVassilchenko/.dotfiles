[zsh]: ~/.zshrc

Move the file to "$HOME"/.config/zsh/.zshrc and export the following environment variable:                          
                                                                                                                    
export ZDOTDIR="$HOME"/.config/zsh                                                                                
                                                                                                                      
You can do this in /etc/zshenv (Or /etc/zsh/zshenv, on some distros).
